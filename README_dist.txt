================================================================================
                          TEDER BOT - 코인원 자동매매
                              USDT/KRW 거래봇 v1.0
================================================================================

이 프로그램은 코인원 거래소에서 USDT/KRW 페어를 자동으로 매매하는 봇입니다.
RSI와 EMA 지표를 기반으로 한 기술적 분석을 통해 자동 거래를 수행합니다.

================================================================================
                                   파일 구성
================================================================================

TederBot.exe    - 메인 실행 파일
config.ini      - 설정 파일 (API 키 및 거래 전략 설정)
logs/           - 로그 파일이 저장되는 폴더
README.txt      - 이 파일 (사용 설명서)

================================================================================
                                 설치 및 설정
================================================================================

1. API 키 설정
   - 코인원(https://coinone.co.kr) 계정에 로그인
   - 마이페이지 > API 관리에서 새 API 키 발급
   - config.ini 파일을 메모장으로 열어 다음 항목 수정:
     ACCESS_TOKEN = 발급받은_액세스_토큰
     SECRET_KEY = 발급받은_시크릿_키

2. 거래 모드 설정
   - 모의거래: DRY_RUN = True (권장, 테스트용)
   - 실거래: DRY_RUN = False (실제 자금 사용, 주의!)

3. 보안 설정 (권장)
   - 코인원 API 설정에서 IP 제한 활성화
   - 거래량 제한 설정
   - 출금 권한은 비활성화

================================================================================
                                   사용 방법
================================================================================

1. TederBot.exe 더블클릭으로 실행
2. 설정 확인 화면에서 정보 검토
3. 모의거래 모드: 바로 시작
   실거래 모드: 10초 카운트다운 후 시작
4. 프로그램 종료: Ctrl+C 키 또는 창 닫기

================================================================================
                                 거래 전략 설명
================================================================================

매수 조건 (모두 충족 시 매수):
- RSI(14) 직전 3봉, 5봉 기울기가 모두 양수
- EMA(20) 직전 3봉 기울기 >= 2.0
- EMA(20) 직전 5봉 기울기 >= 1.5

매도 조건 (하나라도 충족 시 매도):
- 익절: 매수가 + 4원 도달
- 손절: 매수 후 24시간 경과
- 과매수: RSI > 70
- 하락 추세: EMA 기울기 지속적 감소

거래 방식:
- 보유 원화 전액으로 매수
- 매도 1호가로 지정가 매수
- 익절은 지정가, 손절은 시장가 매도

================================================================================
                                   설정 변경
================================================================================

config.ini 파일에서 다음 항목들을 조정할 수 있습니다:

[TRADING] 섹션:
- PROFIT_TARGET: 익절 목표 (기본값: 4원)
- MAX_HOLD_HOURS: 최대 보유시간 (기본값: 24시간)
- RSI_OVERBOUGHT: 과매수 기준 (기본값: 70)
- EMA_SLOPE_THRESHOLD_3/5: EMA 기울기 임계값

[SYSTEM] 섹션:
- CHECK_INTERVAL: 체크 주기 (기본값: 60초)
- LOG_LEVEL: 로그 상세도 (DEBUG/INFO/WARNING/ERROR)

설정 변경 후 프로그램을 다시 시작해야 적용됩니다.

================================================================================
                                   로그 확인
================================================================================

logs/trading_bot.log 파일에서 다음 정보를 확인할 수 있습니다:
- 매수/매도 신호 감지
- 실제 거래 체결 내역
- 손익 계산 결과
- 오류 메시지 및 경고

로그 파일은 자동으로 순환됩니다 (최대 10MB, 5개 백업).

================================================================================
                                   주의사항
================================================================================

⚠️ 실거래 모드 사용 시:
- 반드시 소액으로 테스트해보세요
- 시장 변동성이 클 때는 손실이 발생할 수 있습니다

⚠️ 프로그램 제한사항:
- 코인원 API 호출 제한에 따라 동작합니다
- 갑작스러운 시장 변화에는 대응이 지연될 수 있습니다
- 프로그램 오류 시 포지션이 남을 수 있으니 수동 확인이 필요합니다

================================================================================
                                 문제 해결
================================================================================

Q: API 키 오류가 발생합니다
A: config.ini의 API 키가 올바른지 확인하고, 코인원에서 API가 활성화되어 있는지 확인하세요.

Q: 거래가 실행되지 않습니다
A: 로그를 확인하여 매수 조건이 충족되었는지 확인하세요. 시장 상황에 따라 조건 충족이 드물 수 있습니다.

Q: 프로그램이 멈춥니다
A: 로그를 확인하여 오류 원인을 파악하고, 인터넷 연결 상태를 확인하세요.

Q: 수익이 나지 않습니다
A: 시장 상황과 전략의 한계를 이해하시고, 파라미터 조정이나 전략 변경을 고려해보세요.
